Frequently Asked Questions

How do you manage domain names?

You will register your domain name with a provider of your choice. We will then provide you with the necessary DNS settings to point your domain to the GitHub repository where your website is hosted. This gives you full control of your domain while we handle the technical setup.

What is included in content updates?

Content updates include changes to text, images, and other static content on your website. This is perfect for updating business hours, a new service, or adding a blog post. The number of updates per month varies by tier.

Why use GitHub for hosting?

GitHub Pages provides a secure and free way to host static websites, offering exceptional speed and reliability. It also ensures your website is always up-to-date with the latest version of the code, which we manage for you.
